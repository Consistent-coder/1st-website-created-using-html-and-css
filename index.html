The types of specificity are:

  1. Position
  2. Specificity
  3. Type
  4. Importance

Position:

  1. This is according to the lines more you move down it will be accepted.

  2. Hence C in CSS stands for CASCADING.

Specificity: (least to most)
  • Specifying a style in general:

    • This is used if someone wants to target a larger section of code.

  • Specifying by class:

    • This is used to target elements using class.

  • Specifying by Attributes:

    • This is used to target element/s with same attribute.

  • Specifying by ID's:

    • This is used to target only one element . NOTE: Only one ID can be assigned to one element.

The types according to priority:

  • 1st comes inline css because this target only 1 element.

  • 2nd comes internal css because this targets 1 page.

  • 3rd comes external css because this targets multiple pages.

important tag:

If you mention !important beside any style that style will be taken rest of the other changes will be ignored.
